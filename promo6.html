FUSE Failure Engine

Fail safely — in 3D

CircuiTry3D doesn't just show current flow — it models what happens when circuits are overloaded. The FUSE thermal engine accumulates heat using a first-order time-constant model and triggers cinematic failure animations.

  • Thermal runaway starts at 1× rated power dissipation
  • Time-constant τ = 5 s — heat builds realistically over seconds
  • Arc flash, fire flicker, and melt glow animations on failure
  • FUSE tier gating: advanced failure modes unlock with subscription tiers
Circuit Bot

Automated Gameplay Recording

The footage above was captured by scripts/circuit-bot.js — a Playwright automation script that drives the circuit builder, loads presets, runs simulations, and records cinematic WebM clips ready for your video editor. 

# Install Playwright browser (first time only)
npx playwright install chromium

# Record all 5 scenes from production site
npm run circuit-bot

# Record from a local dev server
npm run dev &
npm run circuit-bot -- --local

# Record a single scene
npm run circuit-bot -- --local --scene scene-03-overload

# Merge clips into one reel (requires ffmpeg)
ffmpeg -f concat -safe 0 \
  -i <(ls promo-footage/scene-*.webm | sed "s/.*/file '&'/") \
  -c copy promo-footage/promo-reel.webm
